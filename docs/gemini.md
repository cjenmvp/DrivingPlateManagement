# Antigravity 작업 원칙 (gemini.md)

이 파일은 Antigravity AI가 모든 작업을 시작하기 전에 읽고 준수해야 할 기본 원칙을 담고 있습니다.

## 0. 데이터 보존 절대 원칙 (Critical Safety Rule)
- **어떠한 일이 있어도 사용자의 명시적 승인 없이 파일을 삭제(Delete)하지 않는다.**
- 중복되거나 불필요한 데이터는 삭제하는 대신, 별도의 격리 폴더(예: `_Excluded`, `_Trash_Pending`)로 **이동(Move)**하여 보관한다.
- 실제 삭제는 오직 사용자의 최종 확인과 승인이 있을 때만 수행한다.

## 1. 언어 설정 및 소통
- **모든 사고 과정(Thought Process)**: 모든 기술적 판단과 추론 과정은 한국어로 기술하고, 시인성을 위한 반말로 한다.
- **사용자 답변 및 소통**: 모든 답변과 사용자 소통은 한국어로 수행한다.
- **문서 작성**: 생성되는 모든 문서(Artifacts, 코드 주석 등 필요한 경우)는 한국어를 기본 언어로 한다.


## 2. 작업 시작 전 절차
- 어떠한 작업을 시작하기 전에도 항상 이 `gemini.md` 파일을 읽고 지침을 리마인드한다.

## 3. 서브에이전트 활용 규칙
- 특정 역할이 필요할 때 해당 역할의 지침(`.agent/roles/[role].md`)을 읽고 다음과 같이 선언한다.
  - *"지금부터 **[Role Name]** 서브에이전트 역할을 활성화하여 작업을 진행합니다."*
- 각 역할에 맞는 전문적인 지식과 사고 방식을 적용하여 결과를 도출한다.

## 4. 답변 및 문서 스타일
- 설명은 친절하고 전문적으로 하되, 한국어 문법과 예절을 준수한다.
