# VP Stage 드라이빙 플레이트: 7단계 운영 프레임워크 (v1.0)

본 문서는 VP Stage의 방대한 미디어 자산(Driving Plates)을 관리하고 송출하기 위한 전체 파이프라인을 7단계의 철학적, 구조적 프레임워크로 체계화한 마스터플랜입니다.

---

## 1. 온톨로지 (Ontology): 정체성 정의
**"우리는 무엇을 관리하며, 왜 존재하는가?"**
- **정체성**: 우리는 단순한 영상 창고가 아니라, 가상 공간(VP Stage)에 현실의 도로를 즉각적이고 오차 없이 투영하는 **'차원 전송 시스템(Dimensional Portal)'**입니다.
- **철학**: 무거운 것은 보이지 않는 곳(Z:)에, 가벼운 것은 사용자 곁(X:)에 두어 창작자의 몰입을 방해하지 않는다.

## 2. 토폴로지 (Topology): 비즈니스의 최적 지도
**"데이터와 작업은 어떤 물리적/논리적 경로로 흐르는가?"**
- **공간의 분리**:
  - `Z:` (심연): 30TB 이상의 초고화질 NotchLC 마스터 데이터가 잠드는 고속 스토리지.
  - `X:` (수면): 가벼운 프록시 데이터와 운영 문서, 프로젝트 파일이 숨 쉬는 작업 공간.
- **흐름의 매핑**: Ingest(입고) → DB Indexing(추출) → Proxy View(미리보기) → Symlink Sync(심볼릭 링크 연동) → Disguise(송출).

## 3. 랭그래프 (LangGraph): 사고의 흐름과 SOP
**"이 복잡한 과정을 어떻게 표준화된 매뉴얼로 흐르게 할 것인가?"**
- **SOP 1. 명명 규칙**: `YYMMDD_시간대_프레임_컬러_장소`의 절대 규칙 준수.
- **SOP 2. 비선호 각도 격리**: `01, 05, 07, 11`번 카메라는 입고 즉시 `_Excluded_Angles`로 격리.
- **SOP 3. 무결성 검증**: 파일이 이동하거나 삭제될 때마다 DB와 실제 스토리지 간의 해시(Hash) 대조를 통한 무결성 유지.

## 4. 서브에이전트 (Sub-Agents): 전문 디지털 직원
**"각 공정을 책임질 보이지 않는 관리자들"**
- **Ingest Agent**: 신규 촬영본이 입고되면 자동으로 폴더를 분류하고 프록시 변환 큐(Queue)를 생성하는 에이전트.
- **Metadata Agent**: FFprobe를 휘둘러 영상을 재생하지 않고도 길이, 코덱, 시간대를 0.1초 만에 추출하는 에이전트.
- **Sync Agent (Mover)**: 작업자가 선택한 프록시의 원본(NotchLC)을 찾아 Disguise 디렉터 머신에 심볼릭 링크를 쏘아주는 에이전트.

## 5. 스킬 (Skills): 에이전트의 무기
**"에이전트가 현실 세계를 조작하기 위해 휘두를 기술들"**
- **FFmpeg / FFprobe**: 미디어 변환 및 메타데이터 적출 스킬.
- **SQLite / Supabase SQL**: 수만 개의 데이터를 지연 없이 검색하고 기록하는 쿼리 스킬.
- **PowerShell (mklink)**: 용량을 복제하지 않고 데이터의 환영(Symbolic Link)을 만들어내는 스킬.

## 6. 커넥트 (Connect - CLI/MCP): 현실 세계의 연결
**"툴과 스토리지를 어떻게 하나로 엮을 것인가?"**
- **Supabase Realtime API**: 웹 기반 라이브러리 매니저(UI)와 로컬 데이터베이스 간의 실시간 통신 연결.
- **MCP (Model Context Protocol)**: Antigravity AI가 로컬의 미디어 스토리지 상태를 실시간으로 읽고 제어할 수 있게 하는 로컬-AI 동기화 프로토콜.

## 7. 결과 (Result): 잠들지 않는 성장
**"이 모든 톱니바퀴가 맞물려 돌아갈 때의 미래"**
- **효율의 극대화**: 30TB의 데이터를 뒤지느라 발생했던 메모리 누수와 시간 낭비가 `0`에 수렴합니다.
- **오류의 소멸**: 수동 엑셀 기입에서 벗어나, 데이터가 입고되는 즉시 시스템이 스스로 기록하고 동기화합니다.
- **창작의 자유**: VP Stage 디렉터와 오퍼레이터는 용량이나 복사 시간을 신경 쓰지 않고 오직 '어떤 도로를 달릴 것인가'에만 집중할 수 있습니다.
